# Comp Sci 337 - Natural Language Processing Project 1

The goal of the project: Given a list of tweets, return host of the event, the award list of the night, the presenters, nominees and the winners of said awards. For the best grades also do extra stuff on top of this.

## Installation

Clone the following repo and then use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.
 
bash
$ git clone https://github.com/canturkay/NLP-Projects
$ cd Project1
$ pip install -r requirements.txt


## Usage
Import the data from canvas and place it into the data folder.

All files and methods named appropriately. The GGresponse object under main.py contains our response class, which is given the data for both 2013 and 2015, and a list of common first names, which are loaded under gg_api.
